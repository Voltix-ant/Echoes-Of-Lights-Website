Librairies pythons

Voici les différentes librairies python utilisées dans notre jeu :

  • Pygame : Cette librairie permet de gérer l'affichage et l'audio du jeu.
  • Threading : Cette librairie permet de séparer les différentes boucles nécessazire au bon fonctionnement du jeu dans des conteneurs différents afin que celles-ci puissent tourner en même temps.
  • AioRTC : Cette librairie permet la connexion P2P entre les deux joueurs grâce à la technologie WebRTC.
  • Requests : Il s'agit d'une libraire permettant de faire des requête HTTP, qu'on utilise notamment pour contacter l'API REST de la base de donnée firebase.
  • D'autres librairies natives de python sont également utilisé, tel que Random pour la génération des codes de partie, Time pour créer des délais lors de l'éxécution, asyncio pour la création de fonctions asynchrones...
